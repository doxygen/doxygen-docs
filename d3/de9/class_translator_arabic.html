"\\usepackage[german]{babel}\n"

or 

"\\usepackage{polski}\n"
 "\\usepackage[latin2]{inputenc}\n"
 "\\usepackage[T1]{fontenc}\n"

The English LaTeX does not use such commands. Because of this the empty string is returned in this implementation.
